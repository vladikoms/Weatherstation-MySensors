Schematic and PCB layout for Diptrace

- U1 - Arduino pro mini 3.3V 8MHz
- U2 - LoRa radio module RA-02 (Semtech sx1278)
- R2 - 4.7k (optional)
- R10, R11 - 10k (optional)
J1...J4 - 6P4C connectors use for connecting sensors:
- J1 Humidity HTU21D
- J2 Temperature DS18B20
- J3 Wind vane AS5600
- J4 Anemometer A3144
